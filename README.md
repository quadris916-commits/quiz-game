# quiz-game
A Python Tkinter quiz game with 25 questions, random 10-question gameplay, 4 multiple-choice options for each question, and live score tracking.
## Features
- Built with Python and Tkinter
- 25 total questions
- 10 random questions each game
- 4 options per question
- Score shown at the top
- Final result screen
- Play again option

## Requirements
- Python 3
- Tkinter (usually comes with Python)

## How to Run
1. Download or clone the repository
2. Open terminal in the project folder
3. Run:

```bash
python quiz_game.py
```
